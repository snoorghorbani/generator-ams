# Generator-ams


## Usage ##

Install `generator-ams`:

```
npm install -g generator-ams
```

Go to your project root directory and with `cd` command:
use folowing command:
```
yo ams:module
```
```
yo ams:state
```
```
yo ams:api
```
```
yo ams:directive
```
